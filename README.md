## How to use
1.下载所有文件的zip包并且解压<br>
2.进入Pjsk_qbot_pub->Go-cqhttp->config.yml确认自己的QQ机器人账号<br>
3.启动go-cqhttp和serv.py(建议使用.bat文件启动)<br>
4.好了<br><br><br>
